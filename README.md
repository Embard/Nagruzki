# Nagruzki Online — расчет нагрузок по СП 30

Статическая онлайн-версия программы **Nagruzki / WaterDemandCalculator** для размещения на GitHub Pages.

## Что внутри

- `index.html` — страница калькулятора.
- `styles.css` — оформление.
- `data.js` — каталог водопотребителей из СП 30.13330.2020, таблица А.2, и таблица интерполяции α.
- `app.js` — логика расчета, таблица строк, панель нормативов выбранной строки, быстрые итоги, свод, печать и выгрузка Word-отчета в табличном виде, как в старой desktop-версии.
- `.nojekyll` — чтобы GitHub Pages отдавал файлы без обработки Jekyll.

## Как выложить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в корень репозитория все файлы из этой папки.
3. Открой `Settings → Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. В `Branch` выбери `main` и папку `/root`.
6. Сохрани настройки.

Через короткое время сайт будет доступен по адресу GitHub Pages репозитория.

## Расчетная логика

Для каждой строки берутся нормативы:

- `q_u,m`, л/сут;
- `q_hr,u`, л/ч;
- `q0`, л/с;
- `q0hr`, л/ч.

Расчет выполняется отдельно для холодной, горячей и общей воды:

- `Qсут = q_u,m × U / 1000`, м³/сут;
- для строк с площадью: `Qсут = q_u,m × F × n / 1000`;
- `Qч,пик = q_hr,u × U`, л/ч;
- `Qч,ср = Qсут / T`, м³/ч;
- `NP = Qч,пик / (q0 × 3600)`;
- `NPhr = Qч,пик / q0hr`;
- `q0экв = ΣQч,пик / (3600 × ΣNP)`;
- `q0hr,экв = ΣQч,пик / ΣNPhr`;
- `q = 5 × q0экв × α`, л/с;
- `qhr = 0.005 × q0hr,экв × αhr`, м³/ч.

Значения `α` и `αhr` определяются линейной интерполяцией по таблице, перенесенной из исходной сборки.

## Отчет

Кнопки **Печать** и **Скачать Word** формируют широкий табличный отчет на альбомном листе: 15 колонок, общий заголовок, нумерация колонок, разделы для холодной, горячей и общей воды, строки водопотребителей и итоговые строки с `q0`, `q0hr`, `α`, `αhr`, `q`, `qhr`.
